122 /**
123  * @brief Insert an item into the designated index of the list.
124  *
125  * This function inserts the item into the designated index of the list and shifts
126  * the trailing items one position to the back-end. The index can be positive or
127  * negative, but its absolute value should be smaller than or equal to the list
128  * size.
129  * The operation supports both forward and backward indexing. For the former one,
130  * the list is traversed from the front-end to the back-end. And for the later one,
131  * the direction is reversed.
132  * To illustrate the behavior, let N denote the list size. Thus for forward indexing
133  * , index 0 and index N-1 point to the head and tail. And for backward indexing,
134  * index -1 and index -N point to the list tail and head.
135  * But no matter which indexing method is applied, the list always grows from the
136  * front-end to the back-end.
137  *
138  * @param self The pointer to the DLinkedList structure
139  * @param item The designated item
140  * @param iIdx The designated index
141  *
142  * @retval SUCC
143  * @retval ERR_NOMEM Insufficient memory space for list extension
144  * @retval ERR_IDX Out of range indexing
145  *
146  * @note The absolute value of the index should be smaller than or equal to the
147  * list size.
148  */
149 int32_t DListInsert(DLinkedList *self, Item item, int32_t iIdx);
